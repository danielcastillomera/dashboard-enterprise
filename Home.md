## Sobre el proyecto

Dashboard Enterprise permite gestionar ventas, compras, inventario, clientes, pedidos y facturación electrónica desde un solo sistema accesible por internet. Está construido con tecnologías modernas y cumple con la normativa del SRI Ecuador.

**Stack tecnológico:** Next.js 16, TypeScript, Tailwind CSS 4, Supabase (PostgreSQL), Prisma ORM.
